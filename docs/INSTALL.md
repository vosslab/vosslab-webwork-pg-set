# INSTALL

## Purpose

This repository is a content pack of WeBWorK `PG` and `PGML` problem files under `problems/`.

## Prerequisites

- A working WeBWorK installation (local, staging, or production).
- Access to the course templates area where custom problem files are stored.

## Install steps

1. Copy this repository (or selected files) into your WeBWorK course templates path.
2. Keep the `problems/` directory structure intact when copying.
3. Open a copied problem from the course and preview it in WeBWorK.

## Notes

- This repo does not include a local WeBWorK runtime.
- If rendering fails, verify required macros are available in your WeBWorK environment.
