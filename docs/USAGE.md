# USAGE

## Primary usage path

1. Pick a problem file under `problems/biochemistry/`.
2. Add the file to a WeBWorK assignment.
3. Preview and test with multiple seeds before publishing.

## Problem organization

- Topic directories are grouped by `topicXX`.
- Many topics include paired variants such as `-matching.pgml` and `-which_one.pgml`.

## Maintenance tips

- Keep new files in the same topic-based layout.
- Use clear, consistent file naming to match existing patterns.
- Re-test randomized behavior after edits.
