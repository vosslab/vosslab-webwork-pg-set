## 2026-02-26

### Additions and New Features

- Added `problems/biochemistry/topic06/chemical_reaction_definition.pgml` - foundational multiple-choice question testing whether students can identify defining characteristics of a chemical reaction and distinguish chemical changes from physical changes, with 8 randomized scenarios covering definitions, examples, biochemistry applications (ATP hydrolysis, glucose oxidation), and common misconceptions
- Added `problems/biochemistry/topic06/enzyme_classification-which_one.pgml` - visual which-one question for the six EC enzyme classes (oxidoreductases, transferases, hydrolases, lyases, isomerases, ligases) with 24 biochemical reactions (4 per class), RDKit.js molecular structures for all reactions, inline cofactor text (NAD+, ATP, FAD, H2O, CO2, etc.) displayed alongside structures, canvas width scaled by sqrt(heavy atom count), color-coded radio button choices, molecular formulas displayed below each structure name, and solution explanations. Replaced non-renderable reactions (starch, acetyl-CoA, tRNA) with visually equivalent alternatives (urea hydrolysis, asparagine synthetase, glutamate-cysteine ligase). Sugar molecules use ring form SMILES (pyranose for glucose, furanose for fructose). Refactored molecule data into a shared `%molecules` lookup hash (34 entries) to eliminate duplicate SMILES/formula/label definitions across the 24 questions
